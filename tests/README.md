# Tests Directory

This directory is reserved for unit tests and integration tests for the diabetes prediction project.

## Future Test Files

- `test_eda.py` - Tests for exploratory data analysis functions
- `test_preprocess.py` - Tests for data preprocessing functions
- `test_model.py` - Tests for model training and evaluation
- `test_fine_tuning.py` - Tests for hyperparameter tuning
- `test_app.py` - Tests for Streamlit app functionality

## Running Tests

Once tests are implemented, run them using:

```bash
pytest tests/
```

Or for a specific test file:

```bash
pytest tests/test_preprocess.py
```
